Klaas Bloem (Yahoo!, FZ10 group, 8 May 2004) writes...

The FZ10 has no noticeable pin cushion distortion at full telephoto {e.g. 12x} but barrel distortion at full wide angle {e.g. 1x} can be a problem.

Open this web page: Barrel Distortion Correction tutorial by Vincent Bockaert. Read the text on this page, download and install the Panorama Tools {see Section } as described. For the correction of barrel distortion at 6mm (full wide angle) use a value of -0.016 for `b', 1.016 for `d'.
